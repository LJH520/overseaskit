# OverseaKit

`@unipus/overseaskit` 是一套为出海项目准备的 React 组件库。它不是通用 UI 皮肤库，而是偏业务能力型的组件集合，优先沉淀在国际化产品、海外活动页、工具型页面和 WebView 容器中反复出现的能力。

当前库已提供：

- `ImageEditor`：图片导入、裁剪、旋转、文字、涂鸦、贴纸、滤镜与保存。

后续可以继续扩展的方向：

- 出海营销活动所需的上传、海报、分享与素材处理组件。
- 适合海外场景的支付、表单、埋点、弹层与引导能力封装。
- 面向 WebView / RN 容器的消息通信、鉴权与桥接辅助组件。

建议先阅读[开发指南](/guide)，再查看具体组件文档。
